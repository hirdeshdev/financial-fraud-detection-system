# 📌 Project Overview

The Financial Fraud Detection System is a Machine Learning-based dashboard that helps identify suspicious financial activities by analyzing transaction details entered by users.

This system provides multiple financial intelligence modules in one integrated dashboard:

- 💳 Credit Card Fraud Detection  
- 🏦 Loan Default Prediction  
- 📊 Customer Risk Scoring  
- 🔎 Transaction Anomaly Detection  
- 📈 Spending Pattern Clustering  

---

# 🎯 Main Objective

To build a smart financial analysis platform that predicts fraud or genuine transactions in real time while also offering additional financial risk analysis tools.
